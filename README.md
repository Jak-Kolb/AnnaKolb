# Anna Kolb Portfolio

This repository contains a simple React application showcasing the artwork of Anna Kolb.

## Development

Navigate to the `anna-kolb-portfolio` folder and install dependencies:

```bash
npm install
npm start
```

The app will be available at [http://localhost:3000](http://localhost:3000).

## Adding Artwork

Image data lives in `anna-kolb-portfolio/src/data/artworks.js`. Add new objects to this array and place the corresponding image files in `anna-kolb-portfolio/public/imgs`.
